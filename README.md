# ETH Based Networks

## About

ETH Based networks library.

## Installation

`npm i eth-based-networks`

## Usage

```js
const networks = require('eth-based-networks');
// Or
import networks from 'eth-based-networks'

// Mainnets
const mainnets = networks.mainnets;

// Testnets
const testnets = networks.testnets;
```
